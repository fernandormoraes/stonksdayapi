# StonksDay 
A WebScraper who get data from marketwatch.

Request example:

    GET http://localhost:1323/TSLA

        "name": "Tesla Inc.",
        "price": "183.75",
        "changePoint": "0.49",
        "changePercentage": "0.27",
        "totalVol": "143.48M",
        "dayRangeLower": "177.14",
        "dayRangeHigher": "183.80",
        "week52RangeLower": "101.81",
        "week52RangeHigher": "384.29"
    }
